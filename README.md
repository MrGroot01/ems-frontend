# 🚀 EMS Pro - Employee Management System

A modern Employee Management System built using React, Django, PostgreSQL, and AWS.

## 🌐 Live Demo

https://ems-frontend-blue-six.vercel.app/login

---

# 📋 Project Overview

EMS Pro is a full-stack web application designed to streamline employee management processes including attendance tracking, leave management, payroll processing, task assignment, notifications, reporting, and learning management.

The application provides separate dashboards for Admins and Employees with secure authentication and role-based access control.
